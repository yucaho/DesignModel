原型模式(Prototype) 用原型实例指定创建对象的种类，并且通过拷贝这些原型创建新的对象

原型模式 其实就是从一个对象再创建另外一个可定制的对象，而且不需知道任何创建的细节。

在Java中。clone()方法是浅复制
浅复制(浅克隆):被复制对象的所有变量都含有与原来的对象相同的值，而所有的对其他对象的
引用仍然指向原来的对象。
换言之，浅复制仅仅复制所考虑的对象，而不复制它所引用的对象。

深复制（深克隆）:被复制对象的所有变量都含有与练来的对象相同的值，那些引用其他对象的
变量将被指向被复制过得新对象，而不再是原有的那些被引用的对象。
换言之，深复制把要复制的对象所引用的对象都复制了一遍
